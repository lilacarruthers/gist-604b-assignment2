## Overview
This repository documents my work for assignment #2 in "Module 2: QGIS Desktop GIS"
of GIST 604B. I had never used QGIS before this, so the projects and results in this
repository are my first time navigating the software. Throughout this assignment, I 
learned about some of the basic functionalities within QGIS including editing projections,
manipulating datasets/files, reworking attributes or styling features.

## Reflection
Throughout this process, I think managing layers, files, and datasets became
more clear as I gained more practice. Initially, the layout of QGIS was very
confusing to me because I hadn't used it before but as the tutorials went on
I was able to find tools and fill in prompts without needing to refer back to 
the tutorial for every step. The main challenge I had was with the Performing
Spatial Queries tutorial. I couldn't figure out how to add another layer to 
a GeoPackage, so I kept having to replace it. All of my layers remained in my
project so I am not sure how this error was impacting the file storage. I 
anticipate that these skills will be very useful when I need to do spatial 
analysis on my mac, which is incompatible with ArcGIS Pro and some other GIS
software.
